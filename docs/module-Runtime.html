<static> on(name, callback, tag)

Adds an event listener for the given event name.
Parameters:
Name Type Argument Description
name String The name of the event to listen to, e.g. 'syntaxError'.
callback function The function to call when the event is fired.
tag String <optional>
 Indicates the event listener set to be attached to. Each tag corresponds to a separate parse of the tree, run in the order that the tag was added. If an event listener is going to modify the tree, a tag must be provided so that it doesn't stomp on the other event listeners.
